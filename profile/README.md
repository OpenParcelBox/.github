# OpenParcelBox

OpenParcelBox is an open ecosystem for smart parcel delivery infrastructure, retrofit parcel boxes, open locks and local-first access control.

The project is currently in the early planning and specification phase.

## What this is about

OpenParcelBox aims to make existing parcel boxes, cabinets, lockers, garage boxes and other containers usable as smart parcel drop-off stations.

The goal is not a closed proprietary parcel box. The goal is an open ecosystem with documented hardware interfaces, local-first firmware, optional self-hosting, optional cloud services and optional integrations.

It starts with retrofit use cases such as older DHL parcel boxes, but the long-term mission is broader: individual parcel boxes can become open infrastructure for homes, apartment buildings, neighborhoods, villages, housing associations, small commercial areas and civic parcel boxes where parcel stations are too rare or unsuitable.

## Core principles

- Local operation first
- No mandatory cloud
- No mandatory Home Assistant
- No single-carrier lock-in
- Open lock and access profiles
- Retrofit-friendly hardware concepts
- Separate rights for delivery and pickup
- Privacy-aware event handling
- EU/CEN-oriented development, especially around CEN/TS 17457:2020
- Open source for everyone, including DIY builders, communities and commercial providers
