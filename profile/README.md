# OpenParcelBox

OpenParcelBox is an open ecosystem for smart parcel delivery infrastructure, retrofit parcel boxes, open locks and self-hosted access control.

The project is currently in the early planning and specification phase.

## What this is about

OpenParcelBox aims to make existing parcel boxes, cabinets, lockers, garage boxes and other containers usable as smart parcel drop-off stations.

The goal is not a closed proprietary parcel box. The goal is an open ecosystem with documented hardware interfaces, local-first firmware, optional selfhosting, optional cloud services and optional integrations.

## Core principles

- Local operation first
- No mandatory cloud
- No mandatory Home Assistant
- No single-carrier lock-in
- Open lock and access profiles
- Retrofit-friendly hardware concepts
- Separate rights for delivery and pickup
- Privacy-aware event handling
- Standards-oriented development, especially around CEN/TS 17457:2020

## Main repositories

- [openparcelbox](https://github.com/OpenParcelBox/openparcelbox): ecosystem planning, documentation, roadmap and project coordination
- [openparcellock](https://github.com/OpenParcelBox/openparcellock): open lock module and lock interface profile

## Planned ecosystem areas
